# Family Budget Management System

This is a simple Node.js + Express + MySQL project scaffold to manage family budgets (income/expenses, categories, monthly budgets, reports).

## Quick start

1. Copy `.env.example` to `.env` and fill your DB credentials.
2. Create the database and tables using `Database/family_budget_schema.sql`.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm start
   ```
5. Open http://localhost:3000
